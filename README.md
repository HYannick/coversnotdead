This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

This is a simple React/Redux musical quizz.

To run it :

```
git clone https://github.com/HYannick/coversnotdead.git coversnotdead
cd coversnotdead
````

And in the folder install both client and server dependencies
```
cd client && npm-install
cd server && npm-install
```

And finally
```
npm start
```

Both client and serer will run thanks to [Concurrently](https://github.com/kimmobrunfeldt/concurrently).
